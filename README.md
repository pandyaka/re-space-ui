# re-space-ui

![Logo](/public/images/logo.png)

### Run in Local
1. Clone this repo
2. Install dependencies
```
$ npm install
```
4. Export API URL
```
$ export API_URL=https://xxx
```
6. Run app
```
$ npm run dev
```

### Build App
1. Build app
```
$ npm run build
```
3. Run app
```
$ npm run start
```
